Netflix Movie Duration Analysis

An in-depth exploration of trends in movie durations on Netflix, using Python and Matplotlib for data analysis.

Project Overview

This project explores the possibility of changing movie durations on Netflix, leveraging skills in exploratory data analysis to uncover trends in the entertainment industry. The analysis tested the hypothesis that the average length of movies on Netflix has been decreasing over time.

Python Code Example


# Importing libraries
import pandas as pd
import matplotlib.pyplot as plt

# Loading data
netflix_df = pd.read_csv("netflix_data.csv")
netflix_subset = netflix_df[netflix_df["type"] != 'TV Show']
netflix_movies = netflix_subset[['title', 'country', 'genre', 'release_year', 'duration']]
short_movies = netflix_movies[netflix_movies['duration']<60]

colors = []
for lab, row in netflix_movies.iterrows():
    if row['genre'] == "Children":
        colors.append('yellow')
    elif row['genre'] == "Documentaries":
        colors.append('brown')
    elif row['genre'] == "Stand-Up":
        colors.append('blue')
    else:
        colors.append('grey')

plt.scatter(netflix_movies['release_year'], netflix_movies['duration'], color=colors)
plt.xlabel('Release year'); plt.ylabel('Duration (min)')
plt.title('Movie Duration by Year of Release')
plt.show()

Results

The analysis confirmed a noticeable trend in the shortening of movie lengths on Netflix. Several factors influencing this trend were identified, including genre shifts, production constraints, and changing viewer preferences.
